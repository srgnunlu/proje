# proje

## Mobil Uygulama

`mobile/` dizini Expo tabanlı React Native akışını içerir. Kaydırmalı içerik kartları ve yapay zekâ entegrasyonu için örnek bir akış sunar.

### Kurulum

```bash
cd mobile
npm install
```

### Çalıştırma

```bash
# Expo geliştirme sunucusu
npm start

# Android veya iOS emülatörlerinde test
npm run android
npm run ios
```

> AI yanıtları için `app.json` içindeki `extra.openAiApiKey` alanını doldurun veya ortam değişkeni olarak `EXPO_PUBLIC_OPENAI_API_KEY` belirleyin.
